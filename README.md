# SQL-e-commerce-table-projects

This SQL project is organized into DDL, DML, and TCL categories. It includes table creation, data insertion, and transaction control operations. The project demonstrates essential database concepts such as constraints, joins, and rollback/commit for hands-on SQL learning and practicing core SQL concepts in a structured, real-world example.

# E-commerce SQL Project

This project contains SQL scripts for:
- Creating tables
- Inserting sample data
- Running analysis queries

## Files Included

### 1. create_tables.sql
Contains SQL commands to create:
- customers
- products
- orders
tables.

### 2. insert_data.sql
Sample data for testing queries.

### 3. analysis_queries.sql
Important SQL queries for:
- Revenue calculation
- Best-selling products
- Customer analysis
- Category sales

## Skills Used
- JOINs
- GROUP BY
- ORDER BY
- Aggregate functions (SUM, COUNT)
- Subqueries
- SQL constraints
